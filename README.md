\# Sahibinden Otomasyon Araçları 



Bu proje, Sahibinden.com üzerinde eğitim ve test amaçlı geliştirilmiş otomasyon scriptlerini içerir. Farklı kütüphaneler (Selenium, DrissionPage, PyAutoGUI) kullanılarak çeşitli bypass yöntemleri test edilmiştir.



\##  İçerik

\- \*\*Hesap Oluşturucular:\*\* Otomatik hesap açma işlemlerini (form doldurma, Cloudflare bypass vb.) gerçekleştirir.

  - \* Not: Hesap güvenliği nedeniyle, e-posta adresine gelen doğrulama kodu (OTP) kullanıcı tarafından manuel olarak girilmelidir.\*

\- \*\*Giriş Botları:\*\* JS Enjeksiyonu ve tarayıcı otomasyonu ile giriş yapma araçları.



\##  Kurulum (Gereksinimler)

Projeyi çalıştırmadan önce gerekli Python kütüphanelerini yüklemelisiniz. Terminali açıp şu komutu çalıştırın:



pip install -r requirements.txt

\# Sahibinden Automation Toolkit 



This project contains automation scripts developed for educational and testing purposes on Sahibinden.com. Various bypass methods have been tested using different libraries (Selenium, DrissionPage, PyAutoGUI).



\##  Contents

\- \*\*Account Creators:\*\* Performs automated account creation processes (form filling, Cloudflare bypass, etc.). 

&nbsp; - \* Note: For security reasons, the OTP code sent to the email must be entered manually by the user.\*

\- \*\*Login Bots:\*\* Tools for logging in via JS Injection and browser automation.



\## ⚙️ Installation (Requirements)

You must install the required Python libraries before running the project. Open your terminal and run:
